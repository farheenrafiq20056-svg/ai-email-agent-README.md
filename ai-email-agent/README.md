## AI Email Agent

**Problem:** Manually handling incoming emails wastes time and leads to delays in response.  

**Solution:** AI Email Agent automatically categorizes emails (Sales, Support, HR, Legal, Finance, Promotions, Operations) and drafts smart replies using GPT.  

**Tools:** GPT (OpenAI), n8n, Gmail API, Google Docs, Automation workflows  

**Features:**
- Classifies incoming emails into categories.
- Generates AI-powered responses automatically.
- Marks emails as read and applies labels.
- Supports multiple departments: HR, Legal, Finance, Customer Support, Sales.
